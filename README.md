# My-linux-dot-files
My linux dot files for myself or anyone who wants to use it

# How to install:
1. Git clone this repository
2. Download the following: waybar, waypaper, hyprland, kitty, rofi, swaync, grim, slurp, swappy, and pywal/pywal16colors
3. Drop all the folders in the git clone folder into your .configs except for wallpapers those go into documents
5. Tweak the settings to your liking and make sure all the folders are correctly mentioned in the configs. 
6. Enjoy
